Detailed Description

Abstract base class providing plot and labeling functionality for functions F:X->Y.

Override mpFX::GetY to implement a function. Override mpFX::GetMinY and mpFX::GetMaxY to provide min and max Y range Optionally implement a constructor and pass a name (label) and a label alignment to the constructor mpFX::mpFX. If the layer name is empty, no label will be plotted.
